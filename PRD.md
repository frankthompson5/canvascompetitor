# AutoCourse OS MVP

## Goal
Generate a structured, self-paced course from a user-entered learning goal.

## Product Vision
A system that converts any learning goal into a structured, self-paced, certifiable course instantly.

## Core User Story
As a self-directed learner, I want to enter a topic and receive a complete course with materials, assessments, and progression requirements so that I can prove real competency.

## Target Users
- Self-directed learners
- Career switchers
- Students supplementing college
- Professionals upskilling
- Lifelong learners

## MVP Features
1. **Topic Input Form**
   - Free-text goal
   - Optional level, duration, goal type
2. **AI-Generated Syllabus**
   - 3–5 modules
   - Lessons and resources per module
3. **Module Execution Experience**
   - One quiz per module
   - One assignment per module
   - One video assessment per module
4. **Locked Progression**
   - Next module unlocks only after requirements are completed
5. **Progress Tracking**
   - Simple completion %
6. **Completion Certificate**
   - Basic PDF for users who complete all module gates

## Product Differentiator
This is **not** a static course marketplace. It is an on-demand course generation + competency enforcement system.

## Productive Friction (Required)
Every module should enforce:
- Cognitive friction (think before proceeding)
- Effort friction (produce work artifacts)
- Accountability friction (video proof)
- Progression friction (mastery gates)

## Non-Goals (MVP)
- Accredited certification
- Peer/community features
- Instructor marketplace
- Advanced proctoring
- Enterprise admin controls

## Success Criteria
A new user can create a course, complete module 1 with required friction, and clearly understand why advancement is gated.
